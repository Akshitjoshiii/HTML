HTML TAGS

  1. < p > Paragraph Tag < /p >

    The < p > and < /p > are the HTML tags and “Paragraph Tag” is the HTML element, i.e. the on-page text. This tag formats any text between the opening < p > tag and the closing < /p > tag as a standard paragraph or main body text.

  2. Bold Tag

    Here the < b > and < /b > are the HTML tags and “Bold Tag” is the HTML element, i.e. the on page text. This tag will format any text between the opening < b > tag and the closing < /b > tag as bold.

  3. < i > Italic Tag < /i >

    Here, the < i > and < /i > are the HTML tags and “Italic Tag” is the HTML element (the on-page text.) This tag will format any text between the opening < i > tag and the closing < /i > tag as italic.

  4. < u > Underline Tag < /u >

    Here the < u > and < /u > are the HTML tags and “Underline Tag” is the HTML element, i.e. the on page text. This tag will format any text between the opening < u > tag and the closing < /u > tag as underlined.
